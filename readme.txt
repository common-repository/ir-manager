Redirect or block execution of multiple third parties Java Scripts, Domains, Sub-domains or Url's for multiple pages of your WordPress website or blog.


== Description ==

With this plugin you will be able to block third party source scripts execution and fill that placements with your website articles by Tags / Category / Latest respectively.
IR ad manager is a free to use Ad management system where you can redirect your url's to homepage or on desire internal or external url's. You can also block executing domains, sub domains or url's on your desire Pages of your website.

**Main Features**

*   Block or redirect third party java scripts/domains/sub-domains and specific url for your Web and AMP pages.

*   Collapse Ad space or Show article by Tags / Category and by-date respectively on violation and desire pages of your website.   

*   This plugin will automatically clear your website cache for Added/effected url's for third party blocking. (Please note IR Manager will purge cache for WP Fastest cache/W3 Total Cache and WP Super Cache only.

*   IR Ad manager will not purge your CDN or any outsource cache mechanism for effected url's for your website.

*   Increase Rev will be managing Google ads blocking on violation pages for publisher monetizing with us.

See more about Increase Rev llc (https://increaserev.com).


== Installation ==

1. Upload the IR Ad Manager plugin to your site via the "Add New" section of the "Plugins" tab.

2. Once the plugin has been uploaded and activate.

3. point your mouse cursor to "Question mark" sign " ? " for each function brief guidline.

4. You will be prompt for an secret key on Top. (Paste and apply your secret key). If you haven't secret key please do contact sales@increaserev.com.


Alternatively, you can also install this plugin right through your WordPress admin dashboard. Simply search for "IR Manager" in the plugin section. WordPress will download and install the plugin for you automatically.

Yes but domain name is required.
